## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/JulianePires/mobix-challenge

# Access
$ cd mobix-challenge

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
